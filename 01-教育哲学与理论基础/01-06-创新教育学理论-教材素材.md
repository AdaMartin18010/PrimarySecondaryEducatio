## 1. 深度学习与项目式学习(PBL)模板

- **项目主题**：____________________
- **核心驱动性问题**：（一个开放、真实、复杂的，能够激发学生探究欲的问题）
  - *示例：《我们如何为学校设计一个更环保的垃圾分类与回收系统？》*
- **最终公开成果**：（学生需要创造什么来展示他们的学习成果？）
  - *示例：一个包含调研数据、方案设计、成本预算和宣传海报的完整提案，并在全校师生大会上进行路演。*
- **关键知识与技能清单**：
  - **学科知识**：__________（如：环保知识、统计学、材料科学）
  - **核心素养**：__________（如：批判性思维、团队协作、公开演讲、创造力）
- **项目里程碑**：
  - **第一周：项目启动** - 提出驱动性问题，头脑风暴，组建团队。
  - **第二周：调查研究** - 收集数据，采访相关人员（如保洁员、后勤老师）。
  - **第三、四周：方案设计与原型制作** - 绘制设计图，制作模型。
  - **第五周：完善与修订** - 同伴互评，专家指导，修改方案。
  - **第六周：成果展示与反思** - 公开展示，项目复盘。
- **评估量规（简版）**：
  - **研究深度 (25%)**：对问题的理解是否全面、深入？
  - **方案创新性与可行性 (30%)**：方案是否有创意且能够落地？
  - **团队协作 (25%)**：团队成员是否高效合作、分工明确？
  - **成果表达 (20%)**：最终展示是否清晰、有说服力？

## 2. 翻转课堂实施清单

- **课前准备（教师）**
  - [ ] **确定翻转内容**：选择适合在线学习的、以知识传递为主的内容。
  - [ ] **制作或筛选微课视频**：视频时长建议在5-15分钟，突出重点。
  - [ ] **设计学习任务单**：明确要求学生观看视频后需要思考的问题或完成的任务。
  - [ ] **设置检查机制**：设计一个简短的在线测验或一个开放式问题，以确保学生完成了课前学习。
- **课中活动（学生）**
  - [ ] **快速答疑**：针对课前测验中的普遍性错误进行讲解。
  - [ ] **深度探究活动**：将课堂时间聚焦于更高阶的活动，例如：
    - 小组辩论
    - 案例分析
    - 实验探究
    - 解决复杂问题
    - 同伴教学
  - [ ] **教师角色**：教师在教室中巡视，作为引导者和资源提供者，对个别小组进行指导。
- **课后延伸**
  - [ ] **成果分享**：学生在平台上分享自己的课堂学习成果（如讨论纪要、作品照片）。
  - [ ] **反思与反馈**：学生对本次翻转课堂的体验进行反馈。

## 3. 课堂游戏化设计小工具

- **工具一：PBL积分系统**
  - **P (Points)** - 积分：完成不同的学习任务可以获得不同积分（如：完成作业+10分，帮助同学+5分）。
  - **B (Badges)** - 徽章：达成特定成就可获得徽章（如："乐于助人徽章"、"全勤徽章"、"最佳提问者徽章"）。
  - **L (Leaderboards)** - 排行榜：设立不同维度的排行榜（如："总分榜"、"进步最快榜"），避免只有学霸能上榜。

- **工具二："知识大富翁"棋盘**
  - **方法**：将一章的知识点设计成一个大富翁棋盘，每个格子代表一个知识点。学生以小组为单位投骰子前进，走到相应格子需回答问题或完成挑战，率先到达终点的小组获胜。

- **工具三："学习续命条"**
  - **方法**：给每个小组设定一个初始"生命值"（如10点）。在进行快速问答或竞赛时，回答错误的小组会扣除生命值。这个工具能瞬间提升课堂的紧张感和参与度。

## 4. 创客教育迷你项目点子库

- **艺术与工程**：**光影画(Light Painting)** - 利用LED灯和相机的长曝光功能，在空中"画"出各种图案。
- **数学与设计**：**3D打印分形** - 学习分形几何，并用3D建模软件设计一个分形（如科赫雪花），然后打印出来。
- **物理与手工**：**棉花糖挑战** - 用20根意大利面、一米胶带、一米细绳和一个棉花糖，在18分钟内搭建一个尽可能高的、能让棉花糖立在顶端的独立结构。这是一个经典的锻炼工程思维和团队协作的活动。
- **编程与音乐**：**水果钢琴** - 利用Makey Makey电路板和Scratch编程，将香蕉、苹果等导电的水果变成一架可以弹奏的钢琴。
- **社会与设计**：**无障碍设施改造** - 观察校园，为残障人士设计一个改进方案（如一个坡道、一个自动门按钮），并制作出模型。

### 1. 混合式/个性化学习工具

#### a. "翻转课堂"四步设计清单

**核心理念**: 确保线上自学与线下探究的有效连接，实现对课堂时间的价值最大化。

| **设计环节** | **核心问题** | **设计要点** |
| :--- | :--- | :--- |
| **1. 线上学习设计 (Pre-Class)** | *学生在家需要看什么、做什么？* | - **微课视频**: 长度控制在5-15分钟，聚焦1-2个核心概念。 - **阅读材料**: 提供清晰、简洁的文本或图表。 - **检查点 (Checkpoint)**: 设计一个2-3道题的在线小测验，或一个开放性问题，用于快速检查理解情况。 |
| **2. 数据分析与教学准备 (Pre-Class)** | *从学生的线上学习数据中，我发现了什么？* | - 查看小测验的后台数据，找到学生的**共性疑难点**。 - 记录下学生提出的有价值的问题。 |
| **3. 课堂导入 (In-Class Opener)** | *如何开启线下课堂？* | - **精准答疑**: 基于第二步的数据，用5分钟时间，精准澄清共性疑难点。 - **激活联结**: 提出一个问题，将线上所学与本节课的线下活动联系起来。 |
| **4. 线下深度活动 (In-Class Main Activity)** | *如何利用宝贵的课堂时间，让学生应用和深化所学？* | - **任务驱动**: 设计一个需要协作、应用、创造才能完成的**核心任务**。 - **活动形式**: 小组讨论、辩论、实验探究、项目制作、案例分析等。 - **教师角色**: 成为巡视的"向导"，为不同小组提供差异化支持。 |

#### b. R.A.F.T. 写作任务卡

**核心理念**: 通过提供不同的角色、对象、形式和主题，为学生提供有意义的、个性化的写作选择，深化对知识的理解。

**模板使用说明**: 教师围绕一个学习主题（如"细胞"），设计出多个R.A.F.T.组合，让学生自由选择一个来完成。

| **主题：细胞** | **R (Role - 角色)** | **A (Audience - 受众)** | **F (Format - 形式)** | **T (Topic - 主题)** |
| :--- | :--- | :--- | :--- | :--- |
| **选择1** | 一个线粒体 | 细胞核 | 一封求职信 | 阐述为什么我是"细胞的能量工厂"，应该获得更多资源。 |
| **选择2** | 一个癌细胞 | 正常的体细胞 | 一段独白 | 我的疯狂增殖计划，以及为什么我认为你们的规则是愚蠢的。 |
| **选择3** | 一位科幻小说家 | 小学生读者 | 一个短篇故事 | 乘坐微缩飞船在人体内进行细胞探险的奇妙旅程。 |
| **选择4** | 一位记者 | 公众 | 一篇新闻稿 | 紧急！新型病毒入侵，正在攻击我们的免疫细胞！ |

### 2. 游戏化学习工具

#### a. 课堂成就徽章设计工坊

**核心理念**: 将评价的权力下放，让学生参与定义"什么能力是重要的"，从而激发内在动机。

**活动流程**:

1. **头脑风暴**: (面向全班提问) "除了考试分数，我们认为在这个班级里，还有哪些行为或能力是值得被认可和庆祝的？" (引导学生思考如：乐于助人、勇于提问、坚持不懈、想法有创意等)
2. **设计徽章**:
    - 将学生分组，为其中2-3个被提名的能力设计**成就徽章**。
    - 设计内容包括：**徽章名称** (如"坚毅之心")、**徽章图案**、**获得条件** (如"连续三次挑战难题不放弃，并寻求帮助")。
3. **展示与投票**: 各小组展示自己的设计，并由全班投票选出本学期要正式使用的课堂成就徽章。
4. **颁发与展示**: 在教室设立一个"荣誉墙"，展示获得徽章的学生和他们的事迹。

#### b. "选择你的冒险"式叙事模板

**核心理念**: 引入"选择"和"结果"这两个游戏核心要素，让学生在互动叙事中进行决策，理解因果关系。

**模板示例 (历史课)**:

> **你的身份：一名19世纪40年代的英国纺织厂童工**
>
> **情境**: 你每天工作14个小时，环境恶劣，工资微薄。今天，你听说镇上有一些工人在秘密集会，要求增加工资和改善工作条件。
>
> **你的选择**:
>
> **A. 参加集会**: 你觉得这是一个改变命运的机会，尽管有被工头发现开除的风险。 -> **(跳转到故事分支A)**
>
> **B. 告发他们**: 你害怕失去工作，决定把这个消息告诉工头，希望能得到一点赏钱。 -> **(跳转到故事分支B)**
>
> **C. 假装没听见**: 你只想安安稳稳地干活，不想卷入任何麻烦。 -> **(跳转到故事分支C)**
>
> ---
>
> **故事分支A**: 在集会上，你听到了关于《人民宪章》的演说，第一次理解了"选举权"对工人的意义...
>
> **故事分支B**: 工头奖赏了你，但其他工人看你的眼神充满了鄙夷，你感到了前所未有的孤立...

### 3. 创客/深度学习工具

#### a. "设计思维"流程海报/工作单

**核心理念**: 为创造性解决问题的过程提供一个清晰、可视化的脚手架。

**海报/工作单内容**:

1. **E (Empathize - 共情)**
    - *为谁设计？他们的真实需求/痛点是什么？*
    - *我如何去观察、访谈、体验他们的处境？*
2. **D (Define - 定义)**
    - *基于我的共情发现，我们需要解决的核心问题到底是什么？*
    - *请用一句话清晰地定义我们的挑战。*
3. **I (Ideate - 构思)**
    - *针对这个问题，我们能想到多少种可能的解决方案？*
    - *鼓励"疯狂"的想法，暂时不做评判。*
4. **P (Prototype - 原型)**
    - *我们如何用最简单、最低成本的方式，将最好的那个想法做成一个可触摸、可测试的模型？*
    - *目的不是"完美"，而是"可交流"。*
5. **T (Test - 测试)**
    - *让我们的目标用户来使用这个原型。*
    - *观察他们的反应，聆听他们的反馈。我们学到了什么？哪些地方需要改进？*
    - *(然后，返回之前的步骤，开始下一轮迭代...)*

---

#### b. 项目式学习（PBL）反思日志

**核心理念**: 促使学生将注意力从"项目成品"转向对"学习过程"本身的监控和反思。

| **PBL反思日志** | **项目名称:** ________ | **姓名:** ____________ | **第 \_\_ 周** |
| :--- | :--- | :--- | :--- |
| **本周目标** | 我/我们小组本周计划完成的主要目标是什么？ |
| **过程记录** | 1. 我在本周主要负责了哪些工作？ 2. 我们遇到了一个重要的挑战/困难是... 3. 我们是如何解决（或尝试解决）它的？ |
| **团队协作** | 我认为我们小组本周合作得最棒的一点是... 如果可以重来，在合作方面我们可以改进的一点是... |
| **个人成长** | 1. 本周，我学到的一个最重要的**知识**是... 2. 本周，我锻炼了一项最重要的**技能**是... (如：沟通、搜索资料、使用XX工具等) 3. 目前，我还有一个**困惑/问题**是... |
