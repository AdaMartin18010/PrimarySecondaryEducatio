## 八、集合与分类国际化知识递归扩展 Sets & Classification: International Recursive Expansion

### 1. 知识点与权威定义 Knowledge Points & Authoritative Definitions

- **集合的定义 Definition of Set**
  - 新加坡教材：A set is a collection of objects.（Primary Mathematics 5A）
  - 美国Common Core：A set is a well-defined collection of distinct objects, considered as an object in its own right.
  - 芬兰教材：集合是可以明确列举其元素的对象的整体。
  - 英国KS3：A set is a group of things that have something in common.
  - 法国初中：Un ensemble est une collection d’objets appelés éléments.
  - 挪威教材：Et sett er en samling av objekter.

- **子集 Subset**
  - 新加坡：A subset is a set whose elements all belong to another set.
  - 美国：If every element of set A is also in set B, then A is a subset of B.
  - 芬兰/英国/法国/挪威教材均有类似定义。

- **交集、并集、补集 Intersection, Union, Complement**
  - 新加坡：The intersection of two sets is the set of elements that are in both sets. The union is the set of elements in either set. The complement is the set of elements not in the set.
  - 美国/英国/芬兰/法国/挪威教材均有类似定义。

- **Venn图 Venn Diagram**
  - 新加坡：A Venn diagram is a diagram that shows all possible logical relations between a finite collection of sets.
  - 美国/英国/芬兰/法国/挪威教材均有类似定义。

### 2. 推理与证明 Reasoning & Proof

- 例题：用Venn图证明A∩B = B∩A（交集的交换律）
  - 新加坡教材：Draw two overlapping circles, label the intersection. Show that the intersection area is the same regardless of order.
  - 美国教材：Use set notation and Venn diagram to show commutativity.

- 例题：判断某个对象是否属于集合
  - 芬兰教材：给出一组对象，判断哪些属于集合A。

### 3. 案例与活动 Cases & Activities

- 新加坡：教室物品分类活动（Classroom object classification）
- 美国：用Venn图解决实际问题（Solving real-life problems with Venn diagrams）
- 芬兰：生活中的集合归类游戏（Grouping objects in daily life）
- 英国：兴趣小组成员归类（Classifying interest group members）
- 法国：集合与元素的法语表达练习（French vocabulary for sets and elements）
- 挪威：家庭成员集合图（Family member set mapping）

### 4. 能力训练 Skills Training

- 判断题：下列哪些是集合？哪些不是？（Which of the following are sets? Which are not?）
- 开放性问题：请用Venn图表示你和朋友的兴趣爱好重叠情况。
- 创新任务：设计一个生活中的集合分类活动，并用Venn图展示。
- PISA风格题型：给定一组数据，要求学生用集合语言和Venn图分析。

## 九、结构与关系国际化知识递归扩展 Structure & Relations: International Recursive Expansion

### 9.1. 知识点与权威定义 Knowledge Points & Authoritative Definitions

- **结构的定义 Definition of Structure**
  - 美国：A structure is an arrangement and organization of interrelated elements in a material object or system.
  - 新加坡：Structure refers to how parts are organized and related to each other.
  - 英国KS3：A structure is the way in which parts are connected or arranged to form a whole.
  - 芬兰/法国/挪威教材均有类似定义。

- **关系的定义 Definition of Relation**
  - 美国：A relation is a way in which two or more concepts, objects, or people are connected.

  - 英国/芬兰/法国/挪威教材均有类似定义。

- **映射的定义 Definition of Mapping**
  - 美国：A mapping is a rule that assigns each element from one set to an element in another set.

- **范畴论思想 Category Theory Ideas**
  - 英国/美国：Category theory studies objects and the relationships (morphisms) between them, focusing on structure-preserving mappings.

## 十、哲学与思辨国际化知识递归扩展 Philosophy & Critical Thinking: International Recursive Expansion

### 10.5. 批判性思维深化国际化递归扩展 Deepening Critical Thinking: International Recursive Expansion
